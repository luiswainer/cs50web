If there are conflict of formats between classes or hashes, for example, h1 has a style, class myfirstclass has another. The priorities go as follows:

  1. Inline styling
  2. id
  3. class
  4. type

Diferent style selectors

  • a,b: Multiple
  • a b: Descendant
  • a > b: Child selector
  • a+b: Adjacent sibling
  • [a=b]: Atribute selector
  • a:b: pseudo class
  • a::B : Pseudoelement
